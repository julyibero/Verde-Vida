# Verde Vida 🌿

Aplicativo web para el vivero **Verde Vida** de María López en Medellín, Colombia.

## Estructura del Proyecto

```
Verde-Vida/
├── client/          # Frontend (HTML + Tailwind CSS + JS)
│   ├── index.html
│   ├── pages/       # Páginas públicas
│   ├── user/        # Páginas de usuario autenticado
│   ├── components/  # Header, Footer reutilizables
│   ├── js/          # Scripts JavaScript
│   └── css/         # Estilos adicionales
├── server/          # Backend (Node.js + Express + PostgreSQL)
│   ├── config/      # Configuración de base de datos
│   ├── models/      # Modelos Sequelize
│   ├── routes/      # Endpoints API REST
│   ├── controllers/ # Lógica de negocio
│   ├── middleware/  # Autenticación, seguridad
│   └── seeders/     # Datos de prueba
└── uploads/         # Archivos subidos por usuarios
```

## Requisitos

- Node.js 18+
- PostgreSQL 14+
